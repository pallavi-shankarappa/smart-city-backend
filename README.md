# Smart City Management System - Backend

A RESTful backend API for managing city complaints and officer assignments.

## 🚀 Features

- User Authentication (JWT based)
- Role Based Authorization (Citizen / Officer)
- Complaint Creation & Tracking
- Complaint Assignment
- Status Management (pending → in-progress → resolved)
- Dashboard Statistics
- Pagination & Filtering
- Input Validation using express-validator
- Rate Limiting
- Centralized Error Handling

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- Express Validator
- Morgan (Logging)
- CORS
